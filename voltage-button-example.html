Documentación del Efecto Voltage

Estructura:

  • .ai-btn-voltage - Contenedor principal que maneja el efecto
  • .voltage-button - Clase que debe aplicarse al botón interno
  • .secondary - Modificador para botones CTA secundarios

Características:

  • Efecto de rayo azul que se mueve horizontalmente al hacer hover
  • Efecto de rayo dorado que se mueve verticalmente al hacer hover
  • Glow effect que intensifica con el hover
  • Sombras internas y externas que cambian con la interacción
  • Desactivación automática en botones disabled
  • Compatible con design responsive

Colores utilizados:

  • Azul principal: var(--accent-blue) #58A6FF
  • Dorado: var(--accent-gold) #FFC777
  • Hover azul: var(--accent-blue-hover) #388BFD
  • Hover dorado: var(--accent-gold-hover) #E0A84C

Border-Radius:

  • AI buttons: border-radius: 12px (configurable)
  • El valor es configurable según las necesidades del proyecto
